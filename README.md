📝 ToDo List

Aplicação desenvolvida durante o curso de React da Rocketseat.

📌 Descrição

A aplicação consiste em um controle de tarefas (To-Do List) que permite ao usuário gerenciar suas atividades do dia a dia.

Funcionalidades

➕ Adicionar uma nova tarefa

🌐 Simular uma requisição HTTP

✅ Marcar e desmarcar uma tarefa como concluída

🗑️ Remover uma tarefa da listagem

📊 Visualizar o progresso de conclusão das tarefas

🚀 Conceitos aplicados

Mesmo sendo uma aplicação simples, foram aplicados diversos conceitos importantes do ecossistema React:

Gerenciamento de estado

Imutabilidade do estado

Listas e keys no React

Uso de props

Componentização

🛠️ Tecnologias utilizadas

⚛️ React

⚡ Vite

🔷 TypeScript

⚡ Quickstart

Clone o repositório:

git clone git@github.com:palhetabraian/ToDo.git

Instale as dependências:

pnpm install

Execute o projeto:

pnpm run dev
